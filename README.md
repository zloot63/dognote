## DogNote

DogNote는 반려견의 건강과 산책 기록을 관리할 수 있는 오픈소스 앱입니다.  
Firebase를 기반으로 Google & Apple 로그인, 강아지 정보, 산책 경로 등을 저장합니다.

### 주요 기술 스택
- Next.js + TypeScript
- Firebase Authentication & Firestore
- TailwindCSS

### 기능
- 강아지 정보 등록 및 수정
- 산책 일정 기록 (GPS 기반 거리 자동 계산)
- 메인 대시보드에서 일정, 건강 요약 확인 등
