# 🐾 DogNote 프로젝트 요약

_반려견과 함께하는 소중한 순간을 기록하고 관리하는 현대적인 웹 애플리케이션_

---

## 🎯 프로젝트 개요

DogNote는 반려견의 산책, 건강, 일상을 체계적으로 기록하고 관리할 수 있는 웹 애플리케이션입니다.
사용자 중심의 직관적인 UI와 안정적인 데이터 관리를 통해 반려견과의 소중한 추억을 더욱 특별하게
만들어줍니다.

### 핵심 가치

- **사용자 중심**: 반려견 보호자의 실제 니즈에 기반한 기능 설계
- **기술적 우수성**: 최신 웹 기술과 엔터프라이즈급 아키텍처
- **접근성**: 모든 사용자가 동등한 경험을 제공 (WCAG 2.1 AA 준수)
- **보안**: 사용자 데이터를 위한 강력한 보안 체계

## 🏗️ 기술 아키텍처

### 프론트엔드

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript 5.x (strict mode)
- **Styling**: Tailwind CSS + Radix UI + shadcn/ui
- **상태 관리**: Zustand + TanStack Query
- **성능 모니터링**: web-vitals

### 백엔드 및 BaaS

- **데이터베이스**: Supabase Postgres
- **인증**: Supabase Auth (Google, Apple OAuth)
- **파일 저장**: Supabase Storage
- **서버리스**: Supabase Edge Functions (계획)
- **호스팅**: Vercel (프론트엔드)

### 개발 도구

- **테스팅**: Vitest + React Testing Library
- **코드 품질**: ESLint + Prettier + Husky
- **CI/CD**: GitHub Actions (Vercel)
- **번들 분석**: @next/bundle-analyzer
